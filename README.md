# IP Address Tracker
Project developed from a Frontend Mentor challenge. The main purpose is to make an IP Address Tracker app integrating different technologies.

## Get Started

### Install
```bash
yarn
```
or
```bash
npm install
```

### Start
```bash
yarn start
```
or
```bash
npm run start
```
